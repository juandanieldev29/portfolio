Next.js migration scaffold

How to run:

1. cd into `nextjs`
2. Install dependencies: `npm install` (or `pnpm install`)
3. Run dev: `npm run dev`

Notes:
- This folder contains a migrated set of components, pages, styles and types from the Astro site. Images and other static assets referenced using absolute paths (e.g. `/juan-small.jpg`) must be present in this app's `public/` folder or at the root server public folder when deployed.
